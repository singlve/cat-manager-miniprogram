# 猫咪健康管家

微信小程序，用于管理猫咪的健康记录、提醒和基本信息。

## 功能

- 猫咪信息管理（添加、编辑、删除）
- 健康记录追踪（洗澡、驱虫、疫苗、体检等）
- 提醒功能（自动排序：逾期 > 7天内 > 未来）
- 数据云同步（微信云开发）

## 技术栈

- 原生微信小程序
- 微信云开发
- WXML + WXSS + JavaScript

## 本地开发

1. 克隆仓库
2. 用微信开发者工具打开项目
3. 配置 AppID
4. 开通云开发并创建集合：cats、health_records、reminders、users
5. 部署云函数：login、checkReminders
