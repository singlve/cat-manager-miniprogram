## 二、猫咪管理

### 2.1 添加猫咪（cat-add）
- [ ] 输入猫咪名 → 选品种 → 保存 → 首页可见
- [ ] 填出生日期 + 领养日期 → 领养不能早于出生（应 toast 拦截）
- [ ] 选「去喵星了」→ 填去喵星日期 → 不能早于出生
- [ ] 领养日期 > 去喵星日期 → 应 toast 拦截
- [ ] 选头像（chooseAvatar）→ 圆形显示
- [ ] 绝育开关可切换
- [ ] 快速连点保存 → 不重复创建

### 2.2 编辑猫咪（cat-edit）
- [ ] 修改名/品种/生日 → 保存 → 详情页反映
- [ ] 改绝育状态 → 详情页显示 "⚥ 已绝育" / "⚥ 未绝育"
- [ ] 改状态「去喵星了」→ 填日期 → 保存 → 首页显示 🌈
- [ ] 日期链校验与 2.1 一致

### 2.3 删除猫咪（cat-detail）
- [ ] 删除确认 → 猫消失 → 返回首页
- [ ] 快速连点删除 → 不报错
