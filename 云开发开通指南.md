# 云开发开通指南

## 第一步：开通云开发

1. 打开 **微信开发者工具**
2. 在左侧项目列表中，**右键项目根目录** `cat-manager-miniprogram/`
3. 选择 **「更多设置」**
4. 在弹出面板中找到 **「云开发」** → 点击 **「开通」**
5. 首次开通会创建一个默认环境（免费配额：1个数据库 + 5GB存储 + 10GB流量/月的套餐二）

## 第二步：获取环境 ID

1. 开通成功后，点击工具栏的 **「云开发」** 图标（☁️）
2. 进入云开发控制台 → 左侧菜单 → **「设置」**
3. 找到 **「环境ID」**（格式类似 `cat-health-xxxxxx`）
4. 复制这个 ID

## 第三步：填入代码

打开项目文件，**全局替换**环境 ID：

```
文件位置：cat-manager-miniprogram/app.js
第 17 行：const envId = 'YOUR_ENV_ID';  // ← 替换这里
```

示例：
```javascript
const envId = 'cat-health-4a5b6c';  // 你的真实环境ID
```

## 第四步：创建数据库集合

云开发控制台 → **「数据库」** → 按顺序创建以下 3 个集合：

| 集合名 | 用途 | 说明 |
|--------|------|------|
| `cats` | 猫咪档案 | 猫咪基本信息、头像 |
| `health_records` | 健康记录 | 洗澡/驱虫/免疫/体检记录 |
| `reminders` | 定时提醒 | 提醒周期设置 |
| `users` | 用户账号 | 注册用户信息（登录功能）） |

> ⚠️ 创建时选择 **「任何用户可读」** 权限，否则会报权限错误

## 第五步：部署云函数（可选，登录功能需要）

如果需要手机号登录功能，需要部署 `getOpenId` 云函数：

1. 在项目根目录创建 `cloudfunctions/getOpenId/` 目录
2. 在目录内创建 `index.js`：
```javascript
// cloudfunctions/getOpenId/index.js
exports.main = async (event, context) => {
  return {
    openid: wx.cloud.getCurrentCloudUserInfo ? null : context.OPENID
  };
};
```
3. 在目录内创建 `package.json`：
```json
{ "name": "getOpenId", "version": "1.0.0" }
```
4. 右键 `getOpenId/` 目录 → **「上传并部署」**

## 云开发 vs 本地模式

开通云开发并填入环境 ID 后，**自动切换**到云端模式：
- 猫咪数据 → 云数据库（永久保存，多设备同步）
- 头像 → 云存储（永久有效，不会过期）
- 提醒 → 云数据库（多设备同步）

未开通时自动使用本地存储（`wx.setStorageSync`），功能不受影响。

## 常见问题

**Q: 报「权限不足」错误？**
A: 在云开发控制台 → 数据库 → 集合设置 → 权限设置，改为「所有用户可读，仅创建者及管理员可写」

**Q: 头像上传失败？**
A: 检查是否开通了云存储配额，免费套餐有5GB存储空间

**Q: 头像显示不出来？**
A: 可能是 `getAvatarUrl` 获取临时链接失败，刷新页面或检查云存储是否欠费

**Q: 多人使用数据会混乱吗？**
A: 目前代码未按用户隔离，所有用户共用同一份数据。如需隔离，需要在每个集合的记录中加上 `_openid` 字段，按用户过滤。我可以帮你加上。
